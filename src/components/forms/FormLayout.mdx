# Form Layout System

The Form Layout system provides a structured, responsive grid for building complex forms. It consists of three main components:

1. **FormSection**: A container with an optional title, description, and header actions.
2. **FormGrid**: A grid container that defines the number of columns (1-4).
3. **FormItem**: A wrapper for individual fields that controls column spanning.

## Usage

```tsx
import { FormSection, FormGrid, FormItem } from '@/components/forms/FormLayout';

<FormSection title="Organization Details">
  <FormGrid columns={4}>
    {/* Spans 2 columns */}
    <FormItem span={2}>
      <Label>Name</Label>
      <Input />
    </FormItem>
    
    {/* Default span 1 */}
    <FormItem>
      <Label>Type</Label>
      <Select />
    </FormItem>
    
    {/* Spans full width */}
    <FormItem span="full">
      <Label>Description</Label>
      <Textarea />
    </FormItem>
  </FormGrid>
</FormSection>
```

## Responsive Behavior

- **Mobile (< 768px)**: All grids collapse to a single column.
- **Tablet (768px - 1024px)**: 
  - 4-column grids become 2 columns.
  - 3-column grids become 3 columns (crowded) or wrap.
  - 2-column grids remain 2 columns.
- **Desktop (>= 1024px)**: Full column layout is applied.
