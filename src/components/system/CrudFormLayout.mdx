# CrudFormLayout

A standardized layout for Create/Update forms. It wraps the form content in a Card and provides a consistent header and footer with Save/Cancel buttons.

## Usage

Use this component to wrap your form fields.

```tsx
<CrudFormLayout
  title="Create Franchise"
  onSave={handleSave}
  onCancel={handleCancel}
>
  <FormFields />
</CrudFormLayout>
```
