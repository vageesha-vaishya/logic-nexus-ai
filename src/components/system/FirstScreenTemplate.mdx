# FirstScreenTemplate

Standardized template for the first screen (index/list view) of any module. It includes a title, description, breadcrumbs, view toggle, and standard action buttons (Import, Export, New).

## Usage

Use this component as the top-level wrapper for your list pages.

```tsx
<FirstScreenTemplate
  title="Franchises"
  description="Manage franchise locations"
  onCreate={() => handleCreate()}
  viewMode="list"
  onViewModeChange={setViewMode}
>
  <YourListContent />
</FirstScreenTemplate>
```
