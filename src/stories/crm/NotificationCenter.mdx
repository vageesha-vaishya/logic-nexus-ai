# Notification Center

Real-time notification list with filtering tabs, mark-as-read, and dismiss actions.

## Optimization

- Virtualized scroll area keeps rendering efficient
- Controlled button interactions to prevent reflows
- Memoized filtered sets by tab

## Accessibility

- Action buttons have ARIA labels
- Tab navigation is keyboard accessible
- Color use keeps contrast above recommended thresholds
