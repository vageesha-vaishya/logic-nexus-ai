# User Manuals (Role-Based)

## Admin
- Tenant setup; franchise management; role assignment
- Branding configuration; API keys; SLA settings

## Franchise Manager
- User provisioning; permissions; reporting
- Data visibility restricted to franchise scope

## Operator
- Create bookings; manage containers/ULDs; issue HBL/AWB
- Update milestones; upload documents; handle exceptions

## Sales
- Create quotes; manage versions; send to customers
- Convert quotes to shipments; limited cost visibility

## Accounts
- Enter charges; approve invoices; reconcile P&L
- Multi-currency handling; ROE management

## Dispatcher (Trucking)
- Assign drivers/vehicles; monitor routes; record POD

## Warehouse
- Receive cargo; put-away; pick/pack; ship; inventory
